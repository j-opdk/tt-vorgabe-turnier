Ein kleines Projekt, um ein Vorgabeturnier auszurichten.
Bis auf eine kleine Bugkorrektur hat ChatGPT das Coding übernommen - danke dafür.
Die Berechnung der Vorgabe basiert auf folgender Berechnung:
